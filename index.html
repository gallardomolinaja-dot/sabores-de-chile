Descubre la Gastronomía Chilena

Tradición, sabor y cultura en cada plato.



Sabores de Chile en una Página Web


Proyecto

Este proyecto en HTML es un pequeño viaje culinario por Chile, donde cada sección es como un plato servido a la mesa. La página combina colores, imágenes y textos para presentar recetas tradicionales como las empanadas, el pastel de choclo y el curanto, mostrando cómo cada región aporta su propio sabor.
Con una estructura clara y bien organizada usando etiquetas básicas de HTML, el sitio no solo enseña a programar, sino que también despierta el apetito. Es una mezcla perfecta entre tecnología y tradición, donde el código se transforma en cultura y cada línea celebra la rica gastronomía chilena.
